# VSCode-settings
This is a collection of shortcut commands for vscode that I have found useful.

### Line deletion

- shift + cmd + K

### Batch change

- shift + cmd + L

### Multiple line selection

- opt + shift + drag

### Duplicate selected lines

- opt + shift + ↑or↓

### Split screens

- cmd + l

### Code jump

- F12
